jq中对于属性操作：

#### 属性操作 新增

```
        var element =  $("p").eq(1)
        element.attr("name",'神仙');
```
#### 属性操作 删除
```
    element =  $("p").eq(1)
    element.removeAttr("name");
```
#### 属性操作 - addClass

```
     element =  $("p").eq(1)
     element.addClass("add hide");

```

#### 属性操作 - removeClass

```
     element =  $("p").eq(1)
     element.removeClass("hide");
```
